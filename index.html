What's inside

Everything you need

Live Preview
Render HTML in a sandboxed iframe or open full-page in a new tab.
Syntax Highlight
Colour-coded tags, attributes, values, and comments.
Auto Format
Beautify and properly indent messy HTML with one click.
Expand & Collapse
Clean up blank lines or compact code for a quick overview.
Import & Export
Load local .html files or any URL, then download your work.
Deep Links
Share app#sample&preview — every action is URL-addressable.
System / Dark / Light
Three-way theme: follow OS automatically, or lock to dark or light.
Zero Dependencies
Pure HTML, CSS, and JS. No build step. No npm. No bundler. Ever.

Quick Start

Up in 30 seconds

terminal 
# Clone the repository
git clone https://github.com/MrLiPx/html-previewer.git
cd html-previewer

# Open directly in your browser — no server needed
open     app.html  # macOS
start    app.html  # Windows
xdg-open app.html  # Linux

# Deploy — Netlify, Cloudflare Pages & Vercel configs included
